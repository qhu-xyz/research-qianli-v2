### Recommendations
1. Do not promote v0002 over v0 for Group A ranking quality; treat this iteration as statistically neutral.
2. Prioritize fixing threshold methodology debt before drawing conclusions from threshold-dependent metrics (REC/CAP/F1/precision).
3. Add explicit schema validation for required base columns before interaction computation and add unit tests that verify exact interaction formulas (not only shape).
4. For next iteration, prioritize seasonality/temporal-shift features targeting persistent weak months (2022-09, 2022-12), since interaction features did not consistently improve discrimination.

### Gate Calibration
Keep current floors for now (only one post-v0 real-data iteration), but current Layer-3 tolerance (0.02) is very loose relative to observed bottom_2 deltas (~0.001–0.002). After 2-3 more real iterations, consider metric-specific tolerance: tighter for stable ranking gates (AUC/AP/NDCG) and looser for high-variance value-capture gates.
